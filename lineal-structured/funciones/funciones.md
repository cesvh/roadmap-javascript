# **Funciones en Javascript**

## **Declarativas**
```js
function hello(value){
    console.log(`El valor es: ${value}`);
}

hello("299");
```

```
El valor es: 299
```

## **Expresivas**
```js
var my_value = function (value){
    console.log(`El valor es: ${value}`);
}

my_value("299");
```

```
El valor es: 299
```
